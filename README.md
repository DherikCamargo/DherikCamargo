# Controle de Despesas

Esta é uma aplicação simples para registrar entradas (receitas) e saídas (despesas) financeiras.
Tudo é feito em uma única página HTML sem necessidade de backend.

## Como usar

1. Abra o arquivo `index.html` em um navegador web.
2. Preencha os campos de título, descrição e valor para cada entrada ou saída e clique em **Salvar**.
3. A lista de entradas e saídas será exibida em tabelas na página, junto com o total de cada uma e o saldo final.

Não há persistência de dados: ao recarregar a página, as informações são reiniciadas.
